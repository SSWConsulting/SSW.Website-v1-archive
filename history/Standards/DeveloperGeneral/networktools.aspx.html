  1. TCPIQ Bandwidth Meter

    2. This bandwidth calculation is measured against the fastest server to your location (it determines at start-up). It is a little application that is run from your computer that provides an accurate bandwidth calculation and then compares against your last submissions. You can also compare your ISP against others in your area or postal code:

    TCPIQ Bandwidth Meter

  2. CNET Bandwidth Meter

    3. This bandwidth calculation is measured against that of CNET's Web Services. Bandwidth may be lower or higher depending on Internet congestion. :

    Bandwidth Meter

  3. Bandwidth Speed Test

    4. This bandwidth calculation is measured on an image download instead of text. The users can select different mirrors as well as different size downloads for a more accurate result.

    Bandwidth Speed Test

    Other Bandwidth testers we have used in the past:

    • Bandwidth Place Speed Test

  4. NetPing

    5. NetPing is a utility with many useful features such as IP Ranges, Uptime, Processor info, Hard drive info and more :

    Figure: NetPing Utility
    NetPing

  5. NetScanTools

    6. NetScanTools is an excellent little utility that is loaded with functionality for testing networks.

    Incorporating all the usual network diagnostic tools in a small and easy to use GUI, NetScanTools makes troubleshooting your network a breeze.

    Figure: NetScanTools
    NetScanTools

  6. NetworkActiv PIAFCTM

    7. NetworkActiv PIAFCTM is a Packet Analyzer tool with a difference. It operates in 3 different modes (Packet Interceptor, HTTP File Interceptor, and Statistical modes), and unlike many other packet analyzers it does not use or require WinPCap or any other third-party extensions.

    NetworkActiv PIAFCTM is not only easy to use, it is also quite powerful as it includes comprehensive packet-filtering capabilities which let you filter packets based on ports, source and destination ip addresses, etc.

    Figure: NetworkActiv PIAFCTM in Packet Interceptor mode
    NetworkActiv PIAFCTM

  7. Monitor file changes

    8. It's really an annoying thing that when files were removed, you don't know who did this. Though Windows offer us an auditing mechanism to record such operations, soon you'll find it's not easy to find out what you need from thousands of such records...
    Now, don't worry, this tool can help you dump specified servers' event logs to your Access or SQL Server database, you can quickly find out when and who touched those files.

    Click Rules To Better Networks for further info.

  8. TeamViewer

    9. It is very common to support clients or help colleagues to resolve their issues. What a fantastic thing if there is a tool you can see the desktop as yours. Also, the person you supported can see what you do and learn from you at the same time.

    TeamViewer can help you with your supporting smoothly. Moreover, show and share your software, PowerPoint presentations etc.

    Figure: TeamViewer

  10. mRemote

    mRemote is a multi-protocol, multi-tab remote connections manager. It allows you to store all your remote connections in a simple and yet powerful interface.
    mRemote supports following list of protocols:

    • RDP (Remote Desktop)
    • VNC (Virtual Network Computing)
    • ICA (Independent Computing Architecture)
    • SSH (Secure Shell)
    • Telnet (TELecommunication NETwork)
    • HTTP/S (Hypertext Transfer Protocol)
    • Rlogin (Rlogin)
    • RAW

    mRemote

  11. Gizmo Drive

    Gizmo Drive is a free application to mount any VHD files including ISO, BIN, CUE and NRG files to a virtual CD-ROM drive.

    Figure: Gizmo Drive
    Gizmo Drive


  12. Virtual Clone Drive

    Virtual Clone Drive is a free software and probably the best virtual drive software that behaves just like a physical CD\DVD drive, however it exists only virtually.

    Virtual Clone Drive
