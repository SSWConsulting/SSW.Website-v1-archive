Microsoft Windows 2000 Sessions

This session is for developers who are taking a first time look at Windows 2000. Fundamentals of Windows 2000 technologies like COM+ will be introduced by building a Windows DNA application from the ground up, line by line, and ending with a functional solution to a simple business process.

Not sure of the difference between COM, MTS & COM+? Don t panic. Along with introducing a slew of new acronyms - IMDB, QC, TSPM, WLBS, CRM, and more - Windows 2000 introduces a variety of rich new application services that build on the foundation laid by COM & MTS. These application services are collectively known as COM+. If you re thinking of building a distributed application, you need to know about COM+.

Microsoft Access 2000 Sessions

This session is for professional developers who are experienced in using previous versions of Access to develop commercial solutions. This product update will highlight compelling new features for the professional developer including sub datasheets, Data Access Pages, new Access Projects (ADPs) for SQL Server and enhancements to the MDB, Jet 4.0 and VBA. Conversion from previous versions of Access and issues like ADO vs. DAO will also be covered. Lessons learnt from the field and top support issues with Access 2000 will complete this session.

Access Projects that work natively with SQL Server 7 and the Microsoft Database Engine (MSDE) are made possible with Access 2000. Professional developers need to know about building these new projects, upsizing from Jet, replication and the traps and pitfalls. This material will show how Access 2000 can be used to build scalable, distributed and mobile solutions. Database licensing for MSDE and SQL Server 7 will also be covered.
