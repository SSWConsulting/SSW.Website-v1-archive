Microsoft Virtualization for Developers by USA expert Brian Randell

  • Date
    • 11/4/11
  • Duration
    • 2 hours
  • Price
    • Free

ONE DAY ONLY

In this session, Brian will discuss the Microsoft virtualization stack and how it can be used in your development shops. Brian will cover features of Hyper-V for use in on your servers, in your development team’s lab, and even production. He’ll also address how virtualization can be used for disaster recovery and high availability. You’ll see how using TFS 2010 with the build feature can do automated build, deploy, and test workflows. Brian will demonstrate how the Lab Management feature of Visual Studio 2010 can help you reduce the developer-tester ping pong game that occurs when bugs are filed using snapshots and IntelliTrace.
