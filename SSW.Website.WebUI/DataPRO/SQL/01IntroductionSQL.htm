Introduction

Welcome to the 'SSW Data PRO!' SQL Users Guide. This help facility is designed to let you quickly and easily start using 'SSW Data PRO!'.

With 'SSW Data PRO!' you can:

    Make changes to tables, fields and relationships without using the Access interface.

    Log these changes to your database so that it "remembers" the changes that have been made.

    Distribute a front end database that will automatically detect whether the backend is compatible and make any changes necessary to make it so.

Use the menu to your left to learn how to use 'SSW Data PRO!'.
