Comments

The main idea of this option is to allow you to place comments. Later when the automatic upgrading is in progress, a message box will be displayed with your comments related to the Change ID at the time you made comments. When this message box displays, Data PRO! temporarily stops. If appropriate, you can switch over to the data and perform the manual change, then switch back and click 'OK' to let 'SSW Data PRO!' continue the upgrading. Here are some examples of comments you might make:
- Stop here and manually delete 'XYZ' from table 'ABC'.
- Release this version to client 'ABC' today.
