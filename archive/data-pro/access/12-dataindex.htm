Index

Index allows you to add or delete indexes for one or more fields of tables and create dual primary key. When deleting indexes you can easily view all fields with that index.



Example

Creating an Index

Let's assume Microsoft searches on the 'SalesTax' field frequently. To run a query fast you want to index it.

  1. Select Index.
  2. Then select Add.
  3. A new window will pop up. Select 'Customers' as Table.
  4. Under Field Name select 'SalesTax' and type 'SalesTax' under Index Name.
  5. Select 'No' in Unique combo box.
  6. Now click the 'Process' button to create the index.

Your Data version is now 3.
