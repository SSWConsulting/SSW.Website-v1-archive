Using TVP’s and XML to Transport Relational Data

Speaker: Jeff Wharton

Developers can now use a number of different methods for transporting relational data from their applications to SQL Server and sometimes choosing the best method(s) to use can be frustrating. Which one is faster? Which one is easier to implement? Can I use a mixture of both?

In this presentation we’ll take a look at how to implement solutions using XML and Table-valued Parameters (TVP’s) and what’s required to switch between the two technologies. We’ll also take a look at some of the pros and cons associated with using XML and TVP’s in your applications and what can be done to reduce the (perceived) performance overheads associated with using said technologies.


About the Speaker:
