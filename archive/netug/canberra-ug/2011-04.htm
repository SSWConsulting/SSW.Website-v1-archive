Canberra .NET User Group - 3 Days to go!

You are cordially invited to the APRIL meetings of the Canberra .NET Developers User Group.

The lunch time meeting is being held on MONDAY 18th April from 12.30pm to 1.30pm.

The evening meeting is being held on MONDAY 18th April from 4.30pm to 6pm.

Agenda:
  • 20 mins - Pizza, Coke and Networking

  • 60 mins - Workflow Foundation 4

    Speaker: Rory Primrose

    WF4 is the latest version of Windows Workflow Foundation that comes with Visual Studio 2010 and the 4.0 .Net framework. This session will provide a brief overview of how WF4 works, how it has evolved from WF3/3.5 and how you can use it in any type of .Net application. The session will continue with authoring custom activities as time permits.
